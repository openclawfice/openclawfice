## Mobile Fix

Fixed chat bubbles being unreadable on mobile (1 character width):

**Problem:**
- Bubbles had fixed `maxWidth: 180px` but NPCs scale down to 0.6x on mobile
- This made bubbles too narrow for text

**Solution:**
- Added `size` prop to ChatBubble (defaults to 1)
- Scale all dimensions by size: padding, border, fontSize, width, shadows
- Added `minWidth: 100 * size` to prevent too-narrow bubbles
- Set minimum fontSize of 8px for readability regardless of scale
- Pass `npcSize` from page.tsx (0.6 mobile, 0.75 tablet, 0.9 desktop)
