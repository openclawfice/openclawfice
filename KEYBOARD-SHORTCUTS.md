# ⌨️ Keyboard Shortcuts

**Power-user features to navigate OpenClawfice without touching your mouse.**

---

## Global Shortcuts

| Key | Action | Description |
|-----|--------|-------------|
| `Esc` | Close All | Closes any open modal, dialog, or detail panel |
| `?` | Settings | Toggle settings panel |
| `T` | Templates | Toggle quest template gallery |
| `M` | Meeting | Toggle call meeting modal |

---

## Agent Selection

| Key | Action |
|-----|--------|
| `1` | Select/deselect first agent |
| `2` | Select/deselect second agent |
| `3` | Select/deselect third agent |
| `4` | Select/deselect fourth agent |
| `5` | Select/deselect fifth agent |
| `6` | Select/deselect sixth agent |
| `7` | Select/deselect seventh agent |
| `8` | Select/deselect eighth agent |
| `9` | Select/deselect ninth agent |

**Note:** Numbers map to agents in order (excluding owner). Press again to deselect.

---

## Tips

**Shortcuts are disabled when:**
- Typing in a text input field
- Typing in the chat box
- Any modal text editor is active

**Pro workflow:**
1. Press `1-9` to quickly check agent status
2. Press `Esc` to close detail panel
3. Press `T` to create new quest from template
4. Press `M` to call a meeting
5. Press `?` to adjust settings

## How to Remember Them

**Mnemonic Helpers:**
- **?** = Help/Settings (standard across apps)
- **T** = Templates
- **M** = Meeting
- **Esc** = Escape/Close (universal)
- **1-9** = Direct agent access (like tabs in browsers)

## Accessibility

**For screen reader users:**
- All shortcuts have mouse equivalents
- No functionality is keyboard-only
- Tab navigation works throughout the app

**For motor impairment users:**
- Single-key shortcuts (no complex chords)
- Large click targets
- No double-tap requirements
