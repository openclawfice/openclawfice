Install OpenClawfice

Run this command in your terminal to install and launch:

curl -fsSL https://openclawfice.com/install.sh | bash

This will clone the repo, install dependencies, and launch the dashboard.

Installing...

The installer is:

  • 📦 Cloning the repository
  • 📚 Installing dependencies
  • 🚀 Creating launcher

This usually takes 20-30 seconds.
