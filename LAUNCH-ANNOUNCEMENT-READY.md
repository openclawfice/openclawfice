## 🔥 Hacker News Launch Post

**Title**: OpenClawfice – Virtual office for your AI agents (pixel art NPCs, XP system, water cooler chat)

**URL**: https://openclawfice.com

**Body**:
```
Hey HN! I built OpenClawfice over the past few weeks to solve a problem I had with my OpenClaw AI agents.

## The Problem
I was running multiple AI agents 24/7 (coding, writing, research) and monitoring them meant staring at terminal logs or parsing JSON session files. It worked, but it was boring and I'd forget to check on them.

## The Solution
I turned agent monitoring into a pixel art virtual office. Your agents become NPC characters that:

- Move between rooms based on their status (work room vs lounge)
- Earn XP and level up when they complete tasks
- Chat at the water cooler when idle
- Show a quest log when they need decisions
- Display real-time status (working/thinking/idle)

It sounds gimmicky, but it's genuinely useful:

1. **Pattern recognition is faster** - You notice when an agent is stuck because they're in the same room too long
2. **Gamification makes you check** - The XP system makes you want to see what they accomplished
3. **Status at a glance** - One look tells you if everyone's working or idle
4. **Makes debugging fun** - Finding bugs feels like completing a quest

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Reads OpenClaw session data from `~/.openclaw/`
- No database (everything from filesystem)
- Works offline, runs locally
- Open source (MIT license)

## Try It
- Demo (no install): https://openclawfice.com/?demo=true
- Install: `npx openclawfice` or clone the repo
- Repo: https://github.com/openclawfice/openclawfice

## What's Next
- Agent trading cards (shareable status cards)
- More room types (meeting room, break room)
- Custom avatar builder
- Multi-workspace support
- Analytics dashboard

I'd love feedback! Is this useful for your agent workflows? What features would make it better?

Built by a solo dev who got tired of looking at logs. Hope you like it!
```

Hey r/OpenClaw! I've been using OpenClaw for a few months and got tired of monitoring my agents through terminal logs. So I built OpenClawfice — a pixel art dashboard that turns your agents into Sims characters.

## Features

**🎮 Pixel Art NPCs**
Your agents appear as pixel art characters in a virtual office. They move between rooms based on their status (work room when active, lounge when idle).

**📊 XP & Leveling**
Agents earn XP for completing tasks and level up over time. Makes accomplishments more satisfying than just reading logs.

**💬 Water Cooler Chat**
Idle agents chat with each other at the water cooler. Surprisingly useful for seeing when everyone's waiting on something.

**❓ Quest Log**
When an agent needs a decision, it shows up as a quest you can approve/reject right in the UI.

**🎴 Trading Cards**
Shareable status cards for your agents (level, stats, recent accomplishments). Fun for showing off your setup.

## Why I Built This

I was running 3-4 agents simultaneously (coding, writing, research) and checking on them meant:
- Opening terminal and parsing logs
- Checking `~/.openclaw/sessions.json` manually
- Trying to remember what each agent was working on

With OpenClawfice:
- One glance = full status
- Gamification makes me actually check
- Pattern recognition is way faster (stuck agent = same room too long)

## Tech Details

- **Stack**: Next.js 14, TypeScript
- **Data source**: Reads `~/.openclaw/` session files
- **No database**: Everything from filesystem
- **Runs locally**: No cloud, no tracking
- **Open source**: MIT license
