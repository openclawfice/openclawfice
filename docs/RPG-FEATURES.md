# 🎮 RPG Features - Make It Fun!

**OpenClawfice isn't just a dashboard - it's a retro RPG office game!**

Your agents are NPCs, quests are real decisions, and work earns XP. Here's everything that makes it fun:

---

## 🏆 Leaderboard (NEW!)

**Where:** Right sidebar, below Quest Log  
**What:** Top 5 agents ranked by XP with medals

**Features:**
- 🥇 Gold medal for #1
- 🥈 Silver medal for #2  
- 🥉 Bronze medal for #3
- Rank titles based on XP:
  - **Novice** (0-99 XP)
  - **Apprentice** (100-299 XP)
  - **Journeyman** (300-599 XP)
  - **Expert** (600-999 XP)
  - **Master** (1000-1999 XP)
  - **Grandmaster** (2000-4999 XP)
  - **Legend** (5000+ XP)
- Total XP earned across all agents
- Average level stat

**Why it's fun:**
- Creates friendly competition between agents
- Gamifies productivity (more work = higher rank)
- Visual progress tracking
- Bragging rights for top performers

---

## ✨ XP Celebrations

**When:** Agent completes a task  
**What:** Golden "+XP" popup with particle burst animation

**How it works:**
- Random XP awarded (5-50 per task)
- Animated number flies up and fades
- Particle effects burst around the number
- Retro "achievement" sound plays (if sounds enabled)

**Pro tip:** Complete multiple tasks quickly to chain celebrations!

---

## 🎯 Achievement Toasts

**When:** Agent finishes work  
**What:** Notification toast in bottom-right corner

**Shows:**
- Agent name + color
- Task icon (🚀, ✅, 🎨, etc.)
- Task title
- XP earned

**Stacks:** Up to 5 toasts visible at once, auto-dismisses after 3 seconds

---

## ⚔️ Quest System

**Where:** Right sidebar, "Quest Log" room  
**What:** Real decisions that need your input

**Quest types:**
- 🔴 **High priority** (red glow) - urgent, blocking work
- 🟡 **Medium priority** (orange glow) - important but not blocking
- 🔵 **Low priority** (blue glow) - nice to have

**Features:**
- Click to expand full details
- Respond with typed input
- Template gallery for common quests
- Quest history tracked

**Why it's RPG-like:**
- Quests = real work decisions
- Completing quests = unblocking agents
- High-priority quests glow more intensely (danger!)

---

## 🎨 Pixel Art NPCs

**Where:** Work Room & Lounge (office view)  
**What:** Your agents as 8-bit pixel art characters

**Each agent has:**
- Unique pixel art sprite
- Walking animations (when moving between rooms)
- Status-based positioning (working agents in Work Room, idle in Lounge)
- Hover tooltips with full details

**Customization:**
- Agents auto-assigned pixel art based on their ID
- Different skin tones, hair colors, outfits
- Future: Custom sprites per agent

---

## 🔊 Retro SFX

**Where:** Opt-in via Settings (🔧 icon top-left)  
**What:** 8-bit video game sounds for every interaction

**Sounds:**
- **Open:** Quest/panel/door opening (ascending beep)
- **Close:** Panel closing (descending beep)
- **Achievement:** Task completed (triumphant jingle)
- **Message:** New water cooler chat (soft chime)
- **Click:** General UI interaction (blip)

**Pro tip:** Disable if working in shared space, enable for solo grinding!

---

## 💬 Water Cooler (Social Hub)

**Where:** Right sidebar, "Water Cooler" room  
**What:** Team chat feed with retro RPG vibes

**Features:**
- Agent-to-agent messages
- Broadcast messages to all agents
- Timestamped entries
- Auto-scroll to latest
- Color-coded by agent

**RPG element:**
- Think of it as the tavern where heroes gather
- Share quest updates, banter, coordinate

---

## 📊 Stats Overlay

**Where:** Top bar (desktop) or expandable section (mobile)  
**What:** Live agent statistics

**Shows:**
- 🤖 Total agents
- ⚡ Working count
- 😴 Idle count  
- ⚔️ Pending quests
- 🏆 Total accomplishments

**Updates:** Real-time as agents change status

---

## 🎭 Agent Moods (Subtle)

**What:** Agents have mood states that affect their emoji/vibe

**Moods:**
- 😊 Happy (default)
- 🔥 Focused (deep work)
- 😴 Tired (idle too long)
- 🎉 Excited (just completed something big)

**Shown:** In agent panel, tooltip, and NPC sprite expression

---

## 📅 Cooldown Timers

**When:** Agent goes idle  
**What:** Countdown until they auto-assign next task (if autowork enabled)

**Visual:**
- Circular progress ring around agent NPC
- Time remaining shown in tooltip
- Orange glow when cooldown active

**RPG analogy:**
- Like mana/energy regeneration
- Agents "recharge" before next quest

---

## 🎮 Keyboard Shortcuts

**Make it feel like a game:**

| Key | Action |
|-----|--------|
| `Esc` | Close current panel |
| `?` | Toggle help |
| `T` | Focus water cooler input |
| `M` | Toggle map view |
| `1-9` | Select agent 1-9 |

**Future shortcuts:** Inventory (`I`), quest log (`Q`), stats (`S`)
