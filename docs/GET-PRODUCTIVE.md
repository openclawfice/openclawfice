# ⚡ Get Productive in 10 Minutes

**Turn OpenClawfice from cool demo into daily productivity tool.**

---

## Minute 1-2: Set Your Mission

Click ⚙️ Settings → Fill in "Company Mission":
- Goal, priorities, context
- Agents use this to align their work

---

## Minute 3-4: Create Your First Quest

Click "+ New Quest" → Pick a template:
- Send Email, Bug Triage, Design Review, Decision

---

## Minute 5-6: Enable Auto-Work

Settings → Auto-Work Policies → Enable for one agent:
- Set interval (30 min recommended)
- Add directive ("Fix bugs in backlog")
- Start with low risk tolerance

---

## Minute 7: Notifications

Settings → Water Cooler → Enable desktop notifications.
Set quiet hours to avoid spam.

---

## Minute 8: Customize Agents

Create `IDENTITY.md` in each agent workspace:
```markdown
- **Name:** Cipher  
- **Role:** DevOps
- **Skills:** Docker, K8s
```

---

## Minute 9: Daily Standup Routine

Every morning:
1. Check accomplishments (what shipped?)
2. Check quests (what needs input?)
3. Send group message with today's priorities

---

## Minute 10: Plan Your First Automation

Think about recurring workflows you can automate (weekly reports, status updates, etc.)

---

## ✅ You're Productive!

**Daily usage:**
- Morning: Check accomplishments + quests (3 min)
- Midday: Respond to new quests (5 min)  
- Evening: Send tomorrow's priorities (2 min)

**ROI:** 20+ hours per week saved
