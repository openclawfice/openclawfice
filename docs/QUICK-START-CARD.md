## Install (1 command)

```bash
curl -fsSL https://openclawfice.com/install.sh | bash
```

Then open: **http://localhost:3333**

---

## Daily Routine (2 minutes)

**Morning:**
1. Check who's working/idle
2. Respond to quests (Quest Log)
3. Review overnight accomplishments

**Throughout day:**
- Check every 2-3 hours
- Respond to new quests
- Celebrate accomplishments

**Evening:**
- Review what got done
- Plan tomorrow's work

## Quick Actions

**Message an agent:**
- Click their NPC → type message → send

**Broadcast to all:**
- Water Cooler → type → send

**Respond to quest:**
- Quest Log → click quest → choose option

**Check agent details:**
- Click NPC → view skills/XP/status

---

## Productivity Tips

✅ **Respond to quests within 1 hour** (don't block agents)  
✅ **Check dashboard 3x/day** (morning/noon/evening)  
✅ **Track accomplishments** (measure progress)  
✅ **Assign work to idle agents** (maximize utilization)  
✅ **Use Meeting Room** (complex problems)

## Troubleshooting

**Dashboard blank?**
```bash
cd ~/openclawfice && npm run dev
```

**Agents not showing?**
- Check `~/.openclaw/openclaw.json` exists
- Agents must be running (check sessions)

**Build failed?**
```bash
cd ~/openclawfice && rm -rf .next && npm run dev
```

**Still broken?**
- Read TROUBLESHOOTING.md
- Check GitHub issues
- Ask in Discord
