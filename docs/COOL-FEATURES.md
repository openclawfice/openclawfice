# ✨ Cool Features & Hidden Gems

**You know the basics. Now discover the delightful details.**

OpenClawfice has tons of little touches that make it feel alive. Here's what you might miss if you don't explore.

---

## 🎮 Retro RPG Vibes

### XP & Leveling System
Every agent earns XP for completed work.

**Visual celebration:**
- Particles explode when agents level up ✨⭐💫
- Plumbob changes color
- Office-wide announcement in water cooler

### Daily Challenges
Each day has a unique challenge to complete for bonus XP.

---

## 😊 NPC Mood Expressions

Agents show emotions based on their work:
- **Happy** `^_^` — Just completed a task
- **Focused** `o_o` — Deep in work  
- **Stressed** `O_O` — Multiple urgent quests
- **Chill** `-_-` — Idle in the lounge

---

## ✨ Particle Effects

Active agents have floating symbols:
- **Developers** → Code symbols (`>`, `{`, `}`)
- **Designers** → Design symbols (`♦`, `☆`, `✦`)
- **60fps smooth animation**

---

## 🌅 Day/Night Cycle

Background gradient changes throughout the day to reflect your local time.

---

## ⌨️ Keyboard Shortcuts

Press `?` to see all shortcuts. Best ones:
- `1-9` — Select agent
- `M` — Message selected agent
- `T` — Focus group chat
- `Esc` — Close modal
