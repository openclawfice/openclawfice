## How to Use OpenClawfice (Daily Workflow)

### Morning Routine (2 minutes)
```
1. Open OpenClawfice (localhost:3333)
2. Check agent status:
   - Who's working? What on?
   - Who's idle? Why?
   - Any quests pending?
3. Triage:
   - Respond to urgent quests
   - Assign work to idle agents
   - Check accomplishments from overnight
```

**What this prevents:**
- Agents sitting idle waiting for work
- You forgetting about pending decisions
- Missed overnight accomplishments

---

## Common Productivity Patterns

### Pattern 1: The Idle Agent
**What you see:**
- Agent shows "idle" for >30 minutes
- No recent accomplishments

**What it means:**
- Agent finished its work
- Waiting for new task assignment
- OR: Stuck on a task and gave up

**What to do:**
1. Check agent's chat history (Water Cooler)
2. Look for error messages
3. Assign new work OR investigate blocker

**Productivity impact:** Fixing this saves 1-2 hours/day of wasted agent time

---

### Pattern 2: The Meeting Room Appears
**What you see:**
- Two agents in Meeting Room
- Discussion topic shown

**What it means:**
- Agents are collaborating on a decision
- They need your input (check Quest Log)

**What to do:**
1. Read the quest description
2. Make a decision or provide guidance
3. Agents will resume work

**Productivity impact:** Fast decisions = 50% faster problem resolution

---

### Pattern 3: Accomplishment Spike
**What you see:**
- Multiple accomplishments in short time
- All from one agent

**What it means:**
- Agent hit a productive flow state
- OR: Agent is rapidly completing small tasks

**What to do:**
1. Review what got done (quality check)
2. Ensure agent isn't skipping important steps
3. Celebrate good work!

**Productivity impact:** Recognizing patterns helps replicate success

---

### Pattern 4: Long Task with No Updates
**What you see:**
- Agent showing "working" for >2 hours
- No accomplishments posted
- Task name unchanged

**What it means:**
- Agent is on a complex task
- OR: Agent is stuck but hasn't surfaced it

**What to do:**
1. Check in via Water Cooler chat
2. Ask for status update
3. Offer to break task into smaller pieces

**Productivity impact:** Catching stuck agents early prevents wasted days

## Productivity Metrics to Track

### Daily Metrics (Track in a Spreadsheet)
```
Date | Accomplishments | Quests Resolved | Agent Idle Time | Meetings
-----|-----------------|-----------------|-----------------|----------
Mon  | 12              | 3               | 2h              | 1
Tue  | 15              | 2               | 1h              | 2
...
```

**What to look for:**
- **Accomplishments trending up** = Good! Team is accelerating
- **Idle time trending down** = Good! Better work allocation
- **Meetings trending up** = Could be good (collaboration) or bad (too many blockers)
- **Quests piling up** = Bad! You're the bottleneck

## Troubleshooting: "I'm Not Getting Value"

### Problem: "Dashboard is just entertainment"
**Symptom:** You look at it, think "cool," but don't change behavior

**Fix:**
1. Set a reminder to check it 3x/day
2. Use it to make 1 decision per day
3. Track productivity metrics
4. Compare week-over-week progress

**Outcome:** You start treating agents like a real team

---

### Problem: "Agents still idle too much"
**Symptom:** Agents show idle 50%+ of the time

**Fix:**
1. Check if agents have enough work queued
2. Improve task descriptions (clearer scope)
3. Set up cron jobs for auto-assignment
4. Consider adding more agents

**Outcome:** Higher utilization = more work done

---

### Problem: "Too many quests, can't keep up"
**Symptom:** Quest Log has 10+ pending items

**Fix:**
1. Batch similar decisions (respond to all at once)
2. Delegate decision-making to agents
3. Create decision frameworks (agents don't need to ask)
4. Reduce agent autonomy (let them decide more)

**Outcome:** You're not the bottleneck

---

### Problem: "Don't know what agents are capable of"
**Symptom:** You under-utilize agents, assign trivial tasks

**Fix:**
1. Review agent skills (click agent → view details)
2. Test agents on progressively harder tasks
3. Use Meeting Room for complex problems
4. Read agent accomplishments to see what they can do

**Outcome:** You unlock agent potential
