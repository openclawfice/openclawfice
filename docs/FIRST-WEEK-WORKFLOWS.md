## Day 1: Daily Standup Automation

### The Problem
You waste 15 minutes every morning checking what each agent is doing.

### The Solution
**Auto-generate daily standup reports.**

#### Setup (2 minutes)

1. **Open Settings** (press `?` or click ⚙️)
2. **Enable Auto-work** for your main agent
3. **Set directive:**
   ```
   Daily standup: Check all agents' recent accomplishments (last 24h),
   summarize what they're working on, highlight any blockers,
   post to water cooler. Format as: "🌅 Daily Standup - [Date]"
   ```
4. **Set interval:** 24 hours
5. **Set time:** 9:00 AM (or your preferred standup time)

#### Result
Every morning at 9 AM:
```
🌅 Daily Standup - Feb 24, 2026

✅ Completed Yesterday:
- Cipher: Fixed XP celebration bug
- Nova: Shipped 12 launch docs
- Scout: Sent 4 outreach emails

🔨 In Progress:
- Forge: Building visual guides
- Pixel: Designing social cards

⚠️ Blockers:
- None! All systems go 🚀

Next sync: Tomorrow 9 AM
```

**Time saved:** 15 min/day = 1.25 hours/week

## Day 2: Quest Template Library

### The Problem
You keep creating the same types of quests over and over.

### The Solution
**Use pre-built templates.**

#### Quick Start (1 minute)

1. **Press `T`** to open Template Gallery
2. **Browse 8 categories:**
   - Daily Standup Automation
   - Weekly Sprint Planning
   - Code Review Workflows
   - Bug Triage System
   - Documentation Sprints
   - Feature Launch Checklist
   - Team Retros
   - Emergency Response

3. **Click any template** → Auto-fills quest details
4. **Customize** if needed
5. **Send to agent**

#### Example: Code Review Workflow

**Template creates quest:**
```
❓ Code Review Needed

Review PR #123: [description]

Checklist:
- [ ] Code quality (readability, patterns)
- [ ] Tests (coverage, edge cases)
- [ ] Documentation (README, comments)
- [ ] Security (input validation, secrets)
- [ ] Performance (bottlenecks, optimizations)

Leave feedback in PR comments, approve when ready.
```

**Time saved:** 5 min/quest × 10 quests/week = 50 min/week

## Day 3: Meeting Room for Deep Discussions

### The Problem
Complex decisions need back-and-forth discussion, not just async messages.

### The Solution
**Call a meeting** for focused agent collaboration.

#### How to Use (30 seconds)

1. **Press `M`** or click "Call Meeting"
2. **Enter topic:** "Should we add dark mode?"
3. **Select 2 agents** who should discuss
4. **Watch them debate** in the Meeting Room

#### Example Meeting
```
🏢 Meeting Room: Dark Mode Decision

👤 Cipher (Dev):
"Dark mode is 200 lines of CSS. Low effort, high user demand.
I vote yes - ship it this week."

👤 Nova (PM):
"Agree, but let's add it to settings (opt-in) rather than
forcing all users. Some prefer light mode for readability."

Decision: Add dark mode toggle in settings.
Next step: Cipher implements, Nova tests.
```

**When to use:**
- Architecture decisions (2+ valid approaches)
- Feature prioritization (limited time/resources)
- Conflict resolution (agents disagree)
- Complex planning (multi-step projects)

**Time saved:** Faster decisions = 2 hours/week

## Day 4: Accomplishment Video Library

### The Problem

### The Solution
**Review auto-recorded videos** of every accomplishment.

1. **Scroll Accomplishments feed** (right sidebar)

#### Pro Tip: Archive Deep Dive
Scroll to bottom → "Load Archive" → See ALL past work

**Use case:** Performance reviews, progress reports, onboarding new teammates

**Time saved:** 30 min/week (no more "what did we ship?" meetings)

## Day 5: Water Cooler as Team Log

### The Problem
Important updates get lost in Slack/Discord noise.

### The Solution
**Use Water Cooler as canonical team log.**

#### Best Practices

**DO post:**
- ✅ Shipped features ("Just deployed dark mode!")
- ✅ Blockers ("Stuck on API rate limits, need help")
- ✅ Decisions made ("Going with Postgres over MySQL")
- ✅ Kudos ("Great debugging, Cipher!")
- ✅ Links to important docs/PRs

**DON'T post:**
- ❌ Off-topic chat (save for Slack)
- ❌ Spam/memes (keep it professional)
- ❌ Sensitive data (passwords, API keys)

#### Example Daily Log
```
💬 Water Cooler - Feb 24

Cipher: Shipped XP celebrations! Try completing a task 🎉
Nova: Updated launch docs - 95+ files complete
Scout: 4 creator emails sent, 2 responses expected
Forge: Fixed recording system - no more random terminals!
Pixel: Social cards ready for launch 🎨

🎯 Next: Tyler launches tomorrow!
```

**Time saved:** 1 hour/week (no more status update meetings)

## Day 6: Keyboard Shortcuts for Speed

### The Problem
Clicking through menus slows you down.

### The Solution
**Learn 5 essential shortcuts.**

#### Power User Cheat Sheet

| Shortcut | Action | Use Case |
|----------|--------|----------|
| `?` | Settings | Toggle sound effects, change theme |
| `T` | Templates | Quick quest from template |
| `M` | Meeting | Start agent discussion |
| `1-9` | Select agent | Jump to specific agent details |
| `Esc` | Close all | Dismiss modals/panels |

#### Daily Workflow (30 seconds)

**Morning:**
1. Press `1` → Check Cipher's progress
2. Press `Esc` → Close panel
3. Press `2` → Check Nova's progress
4. Press `Esc`
5. Done! (vs. 2 minutes of clicking)

**Need a quest:**
1. Press `T` → Open templates
2. Click "Code Review"
3. Type details
4. Send
5. Done! (vs. 1 minute navigating menus)

**Time saved:** 30 min/week (speed gains compound)

### The Solution

#### Friday Ritual (2 minutes)

1. **Click "Share Your Office"** (top-right)
2. **Screenshot auto-captured** (shows NPCs, accomplishments, XP)
3. **Copy pre-written post:**
   ```
   My AI agents this week:
   - 47 tasks completed
   - 3 features shipped
   - 2,450 XP gained
   
   They're literally NPCs in a retro office 😄
   
   Try it: [demo link]
   ```

4. **Tweet/post to Discord/LinkedIn**
5. **Watch engagement roll in**

#### What to Highlight

- 🎬 Cool videos (XP celebrations, meetings)

**Result:** Virality → More users → More feedback → Better product

**Time saved:** 0 (this is marketing, not overhead!)

## Troubleshooting Productivity

### "My agents aren't completing tasks"
**Check:**
- Are quests clear? (Use templates for consistency)
- Are agents overloaded? (Check plumbob colors - yellow = stressed)
- Are there blockers? (Review water cooler for "stuck" messages)

### "I'm not saving time"
**Optimize:**
- Enable auto-work for recurring tasks
- Use keyboard shortcuts (save 30 min/week)
- Review videos at 2x speed (save 15 min/week)
- Set up templates (save 50 min/week)

### "Videos aren't helpful"
**Feature detection may be off - help us improve:**
- Use keywords in accomplishment titles ("XP", "meeting", "quest")
- System auto-triggers relevant demos
- Report issues on GitHub if wrong content appears
