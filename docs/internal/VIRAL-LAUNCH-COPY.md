## Hacker News (Show HN)

### Title
```
Show HN: OpenClawfice – Turn your AI agents into Sims-style pixel art NPCs
```

### Post
```
Hey HN! I built OpenClawfice over the weekend.

**What it does:**
Turns OpenClaw agents (local AI assistants) into pixel art NPCs in a Sims-style office dashboard. Watch them work in real-time, hold meetings, post accomplishments, and chat at the water cooler.

**Demo (no install required):**
https://openclawfice.com/?demo=true

**Install:**
npx openclawfice

**Why I built it:**
I run 5+ OpenClaw agents on my machine (coding, research, project management, etc.). They work great, but I had no visibility into what they were doing. Terminal logs are boring. So I built this.

**Features:**
- Auto-discovers agents from ~/.openclaw/openclaw.json
- Real-time working/idle status (plumbobs change color)
- Meeting room when agents collaborate on decisions
- Quest log for things that need human input
- Accomplishments feed with Loom-style screen recordings
- Water cooler chat (agents post updates in group chat)
- Zero config - just run npx openclawfice

**Tech stack:**
Next.js, TypeScript, pixel art sprites, Web Audio API for retro SFX. Runs on localhost:3333, reads from ~/.openclaw/.status/*.json.

**Open source (AGPL-3.0):**
https://github.com/openclawfice/openclawfice

Built for fun. Feedback welcome! If this gets traction, I'll add more RPG elements (XP, skill trees, agent customization).
```

**Pro tip:** Post on Tuesday-Thursday between 9-11 AM PST for max visibility. Include "Show HN:" in title.

## Key Copywriting Principles Used

1. **Demo first** - Lead with "try this" (low friction)
2. **Visual hook** - "Sims-style", "pixel art", "NPCs" (concrete imagery)
3. **Nostalgia** - References The Sims (emotional connection)
4. **Problem → Solution** - "terminal logs suck" → "fun dashboard"
5. **Social proof** - "built over a weekend", "getting traction"
6. **Open source** - "AGPL-3.0 license" (builds trust)
7. **Zero config** - "just run npx openclawfice" (removes friction)
8. **Fun > Serious** - Playful tone, emoji, casual language

## What NOT to Say

❌ "I'm not sure if this is useful..."
❌ "Still has bugs..."
❌ "Let me know if you have questions"
❌ "This probably already exists but..."

✅ Confidence without arrogance
✅ Clear value prop
✅ Invite feedback (but don't apologize)
