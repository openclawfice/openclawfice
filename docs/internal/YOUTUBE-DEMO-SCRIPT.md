## 🎨 VISUAL NOTES

### Camera Angles
- **You:** Webcam or phone camera, casual setup
- **Screen:** Screen recording at 1080p, 60fps for smooth NPC movement
- **Split screen:** When comparing logs vs OpenClawfice

### Sound Design
- **Logs section:** Silence or boring typing sounds
- **OpenClawfice sections:** Upbeat 8-bit background music (low volume)
- **Level up moment:** 8-bit "power up" sound effect
- **Transitions:** Retro "whoosh" sound effects

### Text Overlays
- Minimal, only for emphasis
- Retro pixel font (Press Start 2P)
- Use sparingly (demo URL, GitHub link)

### Pacing
- **Fast cuts** - Don't linger on any screen more than 5 seconds
- **Show, don't tell** - Let the UI speak for itself
- **Humor** - Lean into the absurdity (agents trash-talking, leveling up)

## 🎬 PRODUCTION TIPS

### Equipment Needed
- Webcam or phone camera
- Screen recording software (OBS, QuickTime)
- Basic video editor (DaVinci Resolve, iMovie, Premiere)
- Microphone (even AirPods are fine)

### Recording Process
1. **Record B-roll first** - All OpenClawfice screen recordings
2. **Record A-roll** - You talking to camera (can improvise from script)
3. **Edit together** - A-roll over B-roll
4. **Add SFX** - 8-bit sounds from freesound.org
5. **Add music** - Royalty-free chiptune from YouTube Audio Library

### Length Targets
- **Hook (0-10s):** MUST grab attention or people skip
- **Value (10s-2min):** Show features fast, keep pace high
- **CTA (2-3min):** Clear next step (demo link, GitHub)

### What NOT to Do
- ❌ Don't explain how it works (show it)
- ❌ Don't read documentation (boring)
- ❌ Don't apologize for quality (confidence)
- ❌ Don't make it a tutorial (make it a story)
