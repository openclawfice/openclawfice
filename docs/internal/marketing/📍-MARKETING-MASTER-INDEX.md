## 🚨 Common Blockers & Solutions

### "Browser control not working"
**Solution**: Chrome extension relay needs attached tab
1. Open Chrome → twitter.com
2. Click OpenClaw extension (🦞)
3. Verify badge shows "ON"

---

### "Web search blocked"
**Solution**: No Brave API key configured
- **Workaround**: Use manual content from READY-TO-POST-NOW.md
- **Fix**: Run `openclaw configure --section web`

---

### "Can't publish to ClawhHub"
**Solution**: Not logged in
1. Run: `clawhub login` (opens browser)
2. Run: `bash scripts/publish-to-clawhub.sh`

---

### "GitHub repo settings need admin"
**Solution**: Bot account only has push access
- Ask DaftMonk (admin) to update
- Or use GitHub web UI with your account

## 💡 Pro Tips

### Maximize Engagement
1. **Post at peak times**: 9-10 AM, 3-4 PM EST
2. **Reply within 10 minutes** of comments (algorithm boost)
3. **Use GIFs > static images** (higher engagement)
4. **Ask questions** to prompt replies
5. **Retweet user content** (build community)

### Go Viral
1. **Tag influencers** when relevant (not spammy)
2. **Quote tweet** high-traffic posts
3. **Jump on trends early** (<1 hour)
4. **Make it shareable** (memes, stats cards, quotes)
5. **Timing matters** (mornings > evenings)

### Build Community
1. **Spotlight users** who share setups
2. **Feature requests = engagement**
3. **Easter eggs** (treasure hunt vibe)
