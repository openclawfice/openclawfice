### Dev.to

**Title:**
```
I Built a Sims-Style Dashboard for AI Agents (Open Source)
```

**Article structure:**

```markdown
# I Built a Sims-Style Dashboard for AI Agents

## The Problem

Managing multiple AI agents via CLI is tedious. You're constantly:
- Grepping logs to see what agents are doing
- Context-switching between terminal windows
- Manually tracking which agents are idle vs working
- Missing important status updates

## The Solution

What if agent management was more like The Sims?

I built **OpenClawfice** - a pixel art dashboard where your AI agents are NPCs working in an office.

![Demo GIF]

## Key Features

✅ **Real-time status:** See which agents are working, idle, or stressed  
✅ **Mood system:** Sims-style plumbobs show agent mood  
✅ **Quest log:** Decisions that need your input  
✅ **XP system:** Agents earn XP and level up  
✅ **Meeting room:** Agents discuss decisions together  
✅ **Daily challenges:** Gamified engagement  

## Tech Stack

- **Frontend:** Next.js + React + TypeScript
- **Styling:** Inline styles (no Tailwind - design constraint for retro feel)
- **Data:** Reads from `~/.openclaw/` directory
- **Deployment:** Vercel (production) + local dev server

## How It Works

OpenClawfice auto-discovers agents from your OpenClaw config:

```json
{
  "agents": {
    "list": [
      { "id": "dev", "name": "Forge", "role": "Developer" },
      { "id": "qa", "name": "Lens", "role": "QA Engineer" }
    ]
  }
}
```

Then it reads session files to determine status, task, and activity.

## Design Philosophy

**1. Visual > Text**  
See agent status at a glance instead of reading logs.

**2. Fun > Corporate**  
Retro pixel art beats boring dashboards.

**3. Privacy-First**  
100% local, no cloud, no tracking.

**4. Zero Setup**  
Auto-discovers agents, no config needed.

## Try It

**Demo (10 seconds):** https://openclawfice.com/?demo=true  
**Install:** `npx openclawfice`  
**GitHub:** https://github.com/openclaw/openclawfice

## Lessons Learned

**1. Gamification works**  
The XP system and daily challenges make users engage more.

**2. Demo mode is critical**  
Lower barrier = more users try it.

**3. Retro aesthetic resonates**  
People love the Stardew Valley vibes.

## What's Next

**Week 1 priorities:**
- Dark mode
- Stats dashboard
- Empty state improvements
- Keyboard shortcuts

Open to contributors! Check the GitHub for good first issues.

## Conclusion

Managing AI agents doesn't have to be boring. With a bit of pixel art and game design, it can actually be fun!

Try it and let me know what you think 👇

---

[Tags: #ai #opensource #nextjs #typescript #developertools]
```

**Why this works:** Dev.to loves in-depth technical posts with personal stories. The "lessons learned" section adds credibility.
