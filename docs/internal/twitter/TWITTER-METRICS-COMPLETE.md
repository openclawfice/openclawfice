## How to Use

### Daily (5 min total)

**Morning (2 min):**
1. Check follower count → write it down
2. Review yesterday's engagement
3. Plan today's activity

**Evening (3 min):**
1. Check follower count → calculate daily change
2. Count activity (replies, tweets, likes)
3. Note any wins (viral replies, threads, mentions)
4. Fill in `twitter-metrics/daily/YYYY-MM-DD.md`

### Weekly (10 min)

1. Calculate totals (growth, activity, engagement)
2. Identify patterns (what works, what doesn't)
3. Adjust strategy (do more of what works, stop what doesn't)
4. Update `GROWTH-DASHBOARD.md`
