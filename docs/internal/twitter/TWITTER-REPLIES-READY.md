## How to Use

1. Find relevant conversation on Twitter
2. Pick reply that matches the topic
3. Copy-paste (edit slightly if needed to fit context)
4. Post

**Quality check:** Make sure it actually fits the conversation. If you have to force it, skip that thread.

---

## Quick Edits for Natural Fit

- Add "yeah" or "totally" at the start if agreeing
- Reference specific part of their tweet: "the part about X is spot on"
- Ask a follow-up question: "are you using Y for that?"
- Share specific metric: "my agents have shipped 47 tasks this week"

**Example:**
```
Original: "we handle this in openclawfice by..."

Edited: "yeah we had the same issue. in openclawfice we solved it by..."
```

Feels more like a real conversation, less like a template.
