## How to Track

### Follower Count (Manual)
1. Go to your Twitter profile
2. Note follower count
3. Record in daily template

**Automation option:** Use Twitter API if available

---

### Engagement (Manual)
1. Check notifications for likes/replies on your posts
2. Count manually or note high-engagement posts
3. Record in daily template

**Automation option:** Twitter Analytics dashboard

---

### Demo Clicks (Analytics)
If you have Google Analytics or Plausible on openclawfice.com:
1. Check referral traffic from `t.co` (Twitter's link shortener)
2. Filter for `/demo` page visits
3. Record in daily template

**Without analytics:** 
- Ask people who mention trying it: "Did you find us on Twitter?"
- Track DMs asking about the demo

---

### Conversion Attribution
**GitHub stars:**
- Check star timeline on GitHub
- Cross-reference with Twitter activity times
- If stars spike after a viral tweet → attribute to Twitter

**Installs:**
- Monitor Discord/DMs for "found you on Twitter"
- Track mentions of OpenClawfice by others on Twitter
- Note when people share screenshots/cards

## Analysis Framework

### What's Working?
Track patterns:
- Which conversation types get the most engagement?
- What time of day gets best response?
- Original tweets vs replies — which drives more followers?
- Technical depth vs casual posts — which performs better?

### What's Not Working?
Track failures:
- Posts with 0 engagement
- Conversations that went nowhere
- Topics that don't resonate
- Times when engagement drops

### Optimization Opportunities
Weekly review:
- Double down on what works
- Stop doing what doesn't
- Test new angles
- A/B test different styles

## Red Flags to Watch

🚨 **Follower growth stalled** (< +5/week)
→ Action: Change content strategy, engage more actively

🚨 **Engagement dropping** (posts getting < 2 likes)
→ Action: Review recent posts, return to what worked before

🚨 **No demo clicks** (0 clicks for 3+ days)
→ Action: Include demo link more often, make CTAs clearer

🚨 **Community not responding** (OpenClaw core ignoring us)
→ Action: Provide more value, be less promotional
